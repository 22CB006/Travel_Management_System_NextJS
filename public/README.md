# Public Assets

## Background Image

Please add a winter mountain landscape image as `hero-bg.jpg` in this directory.

The image should be:
- High resolution (at least 1920x1080)
- A scenic winter/mountain landscape with snow
- Similar to the Figma design reference

You can use the image from the Figma design or find a suitable alternative from:
- Unsplash (https://unsplash.com/s/photos/winter-mountain)
- Pexels (https://www.pexels.com/search/winter%20landscape/)

The current implementation references: `/hero-bg.jpg`
